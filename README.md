# PawPal+

PawPal+ is a pet care planner that helps organize daily tasks for a pet. It lets you add tasks, sort them by time, detect conflicts, and manage recurring tasks.

## Features

- Add an owner and a pet
- Add tasks with:
  - title
  - duration
  - priority
  - time
  - frequency (once, daily, weekly)
- Sort tasks by time automatically
- Detect conflicts when tasks share the same time
- Mark tasks as complete
- Filter tasks by:
  - all
  - completed
  - incomplete
- Automatically recreate recurring tasks when completed

## Running the App
